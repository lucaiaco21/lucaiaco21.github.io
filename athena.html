The Project

What is Athena.

Athena is a GenAI-powered conversational reporting tool built at Accenture for one of the world's largest FMCG companies. It allows business users and executives to query 16M+ rows of Nielsen syndicated sales data using plain English, getting back tables and visualizations instantly WITH no SQL knowledge required.

Showcase 02

Writing SQL that teaches
a model to think.

For a GenAI tool to generate accurate SQL from plain English, it needed examples — high-quality query-answer pairs that demonstrate the business logic and naming conventions

The task required understanding both the data model (how sell-out performance data is structured across SKUs, markets, and periods) and the business questions that the client users actually asked — then translating it in SQL langauge

Example Query 01 — Market Share by Category Training Sample 
-- Business question: "What is MARS market share by category in the last period?"

SELECT
    f.global_subcategory,
    SUM(f.promo_value) AS client_value,
    SUM(t.total_category_value) AS total_value,
    ROUND(
        SUM(f.promo_value) * 100.0 / SUM(t.total_category_value), 2
    ) AS market_share_pct
FROM fact_flat_table_local_dashboard f
JOIN total_category_summary t
    ON f.global_subcategory = t.global_subcategory
    AND f.mars_period = t.mars_period
WHERE f.mars_period = (SELECT MAX(mars_period) FROM fact_flat_table_local_dashboard)
GROUP BY f.global_subcategory
ORDER BY market_share_pct DESC;
Example Query 02 — Promo vs Non-Promo Distribution Training Sample 
-- Business question: "Compare promo and non-promo value for each subcategory"

SELECT
    global_subcategory,
    SUM(promo_value)     AS total_promo_value,
    SUM(non_promo_value) AS total_non_promo_value,
    ROUND(
        SUM(promo_value) * 100.0 /
        NULLIF(SUM(promo_value) + SUM(non_promo_value), 0), 1
    ) AS promo_mix_pct
FROM fact_flat_table_local_dashboard
WHERE mars_period = (SELECT MAX(mars_period) FROM fact_flat_table_local_dashboard)
GROUP BY global_subcategory
ORDER BY promo_mix_pct DESC;
